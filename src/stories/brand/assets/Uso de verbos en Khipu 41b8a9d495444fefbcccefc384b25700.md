# Uso de verbos en Khipu

### Otros verbos de uso común en pagos digitales

| Concepto | Antónimo | *Observaciones* |
| --- | --- | --- |
| Ingresar | Salir |  |
| Iniciar […] | Cerrar […] | *DEBE explicitar la acción.
EJ: Iniciar sesión / Cerrar sesión* |
| Continuar | Volver […] | Continuar NO DEBE ser la última acción de un flujo.

*‘Volver’ DEBE incluir el destino.
EJ: Volver al sitio de origen* |
| Autorizar […] | Rechazar […] | *DEBEN explicitar la operación referida 
EJ: Autorizar pago / Rechazar pago*
 |
| Comenzar […] | Cancelar […] | *DEBEN explicitar la operación referida 
EJ: Comenzar suscripción
Cancelar suscripción* |
