Buttons (5 Variants)

Sistema de botones según jerarquía visual 

<!-- Primary - Púrpura sólido -->
<button class="kds-btn kds-btn-primary">
    Primary Button
    <span class="kds-icon"><i class="material-symbols-outlined">arrow_forward</i></span>
</button>

<!-- Secondary - Azul sólido -->
<button class="kds-btn kds-btn-secondary">Secondary Button</button>

<!-- Outlined - Borde púrpura -->
<button class="kds-btn kds-btn-outlined">Outlined Button</button>

<!-- Text - Transparente -->
<button class="kds-btn kds-btn-text">
    <span class="kds-icon"><i class="material-symbols-outlined">arrow_back</i></span>
    Text Button
</button>

<!-- Outlined White - Para fondos oscuros -->
<button class="kds-btn kds-btn-outlined-white">Outlined White</button>

<!-- Disabled state -->
<button class="kds-btn kds-btn-primary" disabled>Disabled</button>

Usage Notes

  • Primary: Acción principal (púrpura #8347AD)
  • Secondary: Acción secundaria (azul #3B82F6)
  • Outlined: Acción terciaria (borde púrpura)
  • Text: Navegación (transparente)
  • Outlined White: Solo para fondos oscuros/púrpura
  • Icons: Usa <span class="kds-icon"> para envolver iconos
